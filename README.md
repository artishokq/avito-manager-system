# Система управления объявлениями для модерации (Avito)
### Тестовое задание для стажёра Frontend (осенняя волна 2025)

[Оригинальное задание](https://github.com/avito-tech/tech-internship/blob/main/Tech%20Internships/Frontend/Frontend-trainee-assignment-autumn-2025/Frontend-trainee-assignment-autumn-2025.md)

Приложение для модераторов объявлений Avito: просмотр ленты объявлений, модерация и статистика работы.

## Запуск

### 1. Клонирование

```bash
git clone https://github.com/artishokq/avito-manager-system.git
cd avito-manager-system
```

### 2. Бэкенд

```bash
cd tech-int3-server
npm install
npm start
```

API (по умолчанию): `http://localhost:3001/api/v1`  
Основные маршруты: `/ads`, `/ads/:id`, `/ads/:id/approve|reject|request-changes`, `/stats/*`.

### 3. Фронтенд

```bash
cd ..
npm install
npm run dev
```

Откроется на `http://localhost:3000`.

## Функционал

### Лента объявлений (`/list`)

- Список объявлений с краткой информацией.
- Фильтры: статус, категория, цена, текстовый поиск.
- Сортировка (поле + направление).
- Пагинация.
- Переход на детальную страницу по клику.
- Progress bar для загрузки
- Анимации при нажатии на карточки объявлений
- Анимации при наведении/нажатии на некоторые кнопки

**Горячие клавиши:**

- `/` — фокус на поле поиска.

### Страница объявления (`/item/:id`)

- Полная информация по объявлению.
- Информация о продавце.
- История модерации.
- Действия модератора: одобрить, отклонить, отправить на доработку.
- Кнопки «Предыдущее» / «Следующее» объявление (по текущему списку).
- Progress bar для загрузки

**Горячие клавиши:**

- `←` — предыдущее объявление.  
- `→` — следующее объявление.

### Статистика (`/stats`)
Доступ по ссылке `http://localhost:3000/stats`

- Переключатель периода: Сегодня / 7 дней / 30 дней.
- Карточки метрик:
  - проверено объявлений (отдельно за сегодня, неделю, месяц);
  - доля одобренных;
  - доля отклонённых;
  - среднее время модерации в формате `М мин S сек`.
- Графики:
  - активность по дням;
  - распределение решений;
  - распределение по категориям.
